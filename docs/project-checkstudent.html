About

소규모 학원 강사분들에게 편리한 학생 관리를 지원하는 어플리케이션

기간

Coder's High iOS App Design to Develop Course(2020.07~2020.08)

주요 기능

  • Home 화면

    메인화면에서는 오늘 사용자가 진행하는 수업, 그리고 관리하는 모든 수업에 관한 contact을 진행할 수 있습니다.

  • 출석체크 화면

    홈 화면에서 바로 이동하여 해당 날짜의 출석체크를 진행할 수 있습니다.

  • 문자메세지 화면

    해당 수업의 모든 학생들에게 문자메세지를 보낼 수 있습니다.

  • Class 화면

    자신이 관리하는 수업을 보여주고 추가 또는 삭제할 수 있습니다.

  • Student 화면

    자신의 수업내의 학생 목록을 보여주고 Search할 수 있으며 학생의 정보를 열람할 수 있습니다.
